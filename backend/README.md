
## Crear ambiente virtual de python
```
python3.7 -m venv venv
```

### Activar ambiente virtual
```
source venv/bin/activate
```

### Instalar dependencias
```
pip install -r requirements.txt
```

### Correr aplicación
```
python app.py
```

### Detener aplicación
```
ctrl + c
```

### Desactivar ambiente virtual
```
deactivate
```
